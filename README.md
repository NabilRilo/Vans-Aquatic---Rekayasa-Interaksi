# Vans-Aquatic

## Deskripsi Aplikasi
Aplikasi belanja online untuk memesan ikan hias dan kebutuhan akuatik dengan fitur keranjang, checkout, notifikasi, chat, dan pengelolaan akun. Dirancang agar pengguna dapat berbelanja dengan cepat, mudah, serta mendapatkan pengalaman yang lebih interaktif melalui fitur favorit, inbox, serta informasi produk yang lengkap.
